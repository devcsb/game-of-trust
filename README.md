# 신뢰의 게임 (IPD 신뢰 게임)

반복 죄수의 딜레마를 캐릭터와 대결하며 배우는 미니 게임. 협력과 배신, 통신 오류, 그리고 "왜 협력이 세상 전체에 이로운가"를 직접 플레이하며 깨닫는다. 서버 없이 브라우저에서 전부 계산하고 GitHub Pages로 배포된다.

플레이: https://devcsb.github.io/game-of-trust/

## 어떤 게임인가

매 라운드 **예측 → 선택 → 동시 공개 → 정산** 4단계로 일곱 AI 캐릭터와 차례로 대결한다. 상대의 다음 수를 먼저 읽고(예측), 내 수를 정하고(커밋), 카드가 동시에 뒤집히는 순간 결과를 확인한다. 각 스테이지가 한 가지 교훈을 준다.

| # | 상대 | 교훈 |
|---|---|---|
| 1 호구 | 항상 협력 | 착한 상대는 이용당한다 (유혹) |
| 2 악당 | 항상 배신 | 무조건 협력하면 호구된다 (방어) |
| 3 거울 | 직전 수를 따라함 | 되갚는 상대에겐 협력이 남는 장사 |
| 4 복수귀 | 한 번 배신 = 영구 보복 + 통신오류 | 한 번의 오해가 영원한 파탄 |
| 5 관대한 거울 | 가끔 용서 + 통신오류 | 용서가 신뢰를 되살린다 |
| 6 변덕쟁이 | 이기면 유지, 지면 전환 + 통신오류 | 결과에 반응하는 상대 |
| 7 동전 | 완전 무작위 | 무작위엔 전략이 통하지 않는다 (운) |

학술 용어는 쓰지 않는다. noise는 "통신 오류"로, 협력의 사회적 가치는 "공동 수확"으로 보여준다.

## 안개의 세계 (피벗 모드)

캠페인이 정체가 공개된 상대 한 명을 차례로 상대한다면, 안개의 세계는 정반대다. 상대의 스탠스가 숨겨진 **혼합 개체군**에 나의 **기본 전략 하나**를 들고 들어가, 불확실성 속에서 수확을 극대화한다. 정해진 정답을 따라가는 게 아니라, 단서를 읽어 세계마다 다른 최적 스탠스를 찾아내는 게임이다.

| 세계 | 숨은 단서 | 교훈 |
|---|---|---|
| 🌫️ 안개 낀 장터 | 협력이 짙고, 드물게 칼이 섞임 | 받은 대로 갚는 게 남는 장사 |
| 🏜️ 굶주린 변경 | 배신 유혹이 크지만 원한이 깊음 | 유혹에 넘어가면 영원히 등 돌림 |
| 🌁 메아리 협곡 | 말이 자주 어긋남 (노이즈 20%) | 엄격함은 독, 약간의 용서가 악순환을 끊음 |
| 🌪️ 요동치는 들판 | 내 선택이 다음 들판을 빚음 (진화) | 너그러운 보복이 내가 거둘 땅을 비옥하게 함 |

복잡성은 네 축으로 만든다. **숨은 혼합 개체군**(매 조우마다 분포에서 상대 추출), **변동 보수 행렬**(세계마다 T/R/P/S가 다름), **불확실한 라운드 수 + 통신 오류**, **진화하는 세계**(replicator dynamics에 내 전략을 섞어 다음 세대 분포를 바꿈). 채점은 절대 점수가 아니라 **이론적 최선 대비 효율**이다. 빌더 카드 공간 16개 기본 전략을 brute-force로 평가해 그 세계의 최선을 앵커로 잡고(세계당 한 번만 계산해 메모이즈), 내가 그 몇 %를 거뒀는지로 별을 매긴다.

조우는 **안개 읽기 베팅**이다. 정체가 드러나기 전, 흐릿한 실루엣과 직전까지의 흐름만 보고 "이 상대는 협력적인가 적대적인가"를 먼저 읽는다. 안개가 걷히면 캠페인과 같은 **표정 SVG 아바타** + 행동 트레이스 + 정체가 드러나고, 읽기가 맞으면 🔮통찰이 쌓이고 연속 간파에 보너스가 붙는다(80% 이상이면 독심술사). 정해진 스탠스를 따라가는 게 아니라, 추상적 상황을 읽어 이익을 얻는 루프다.

## 게임 요소

- **예측(간파) 시스템**: 매 라운드 상대의 다음 수를 먼저 예측한다. 적중하면 🔮통찰이 쌓이고, 3연속 간파마다 보너스. 채점은 상대의 "의도" 기준이라 통신 오류가 끼면 "읽기는 정확! 전파만 어긋났어요"라는 순간이 생긴다 (의도 ≠ 전달).
- **통찰 경제**: 점수와 완전히 분리된 자원. 👁엿보기(상대 의도 미리 보기, 그 라운드는 간파 적립 없음)와 📡재전송(통신 오류로 뒤집힌 내 수를 3초 안에 정정 — 복수귀의 도화선을 끌 수 있는 유일한 방법)에 소비한다.
- **동시 공개 연출**: 카드 입장 → 드럼롤 → 동시 플립 → 글리치(노이즈로 뒤집힌 쪽) → 간파/빗나감 스탬프. 탭하면 스킵.
- **스테이지 시그니처 텔**: 호구의 열린 손(카드 상시 공개), 악당의 도발 말풍선(말 ≠ 행동), 거울 속 잔상(내 직전 수가 비침), 복수귀의 도화선(점화 시 화면 흔들림 + 저음), 관대한 거울의 용서 카운터, 변덕쟁이의 기분 풍향계, 동전의 적중률 미터(~50%에 머무는 것 자체가 교훈).
- **전략 공방**: 카드 3슬롯(첫 만남 / 협력받으면 / 배신당하면)으로 나만의 전략을 조립한다. TFT와 동치 조합을 만들면 "거울을 재발명했어요!" 토스트 — Axelrod 원칙을 조립하며 스스로 발견한다. 즉석 스파링(vs 거울/악당)이 카드를 바꿀 때마다 갱신된다.
- **진화의 정원 출전**: 내 전략이 7번째 종으로 replicator dynamics에 참가한다. 실행 전 "60세대 후 번성/생존/멸종?" 예측을 강제하고, 멸종하면 사인(死因) 분석 한 줄이 나온다 — 죽음도 수업이 된다.
- 표정이 바뀌는 SVG 캐릭터(협력하면 웃고, 배신당하면 화남)
- **공동 수확 HUD**: 두 점수의 합(사회후생)을 목표선과 함께. 협력은 6, 일방배신은 5(손실), 상호배신은 2.
- **공동 정원**: 라운드마다 셀이 협력=만개, 배신=시듦으로 누적 (카드가 뒤집히는 순간 갱신)
- **자원 교환 연출**: 협력=공동 풀로 흐름, 배신=강탈+균열
- 신뢰 게이지, 연속 협력 콤보(콤보가 쌓일수록 화음이 올라가는 사운드), 간파 스트릭 콘페티, Web Audio 합성 효과음(음소거 토글)
- **규칙 온보딩**(payoff 표), 별 1~3개·잠금 해제·진행 저장(localStorage)
- **플레이 스타일 분석**: 클리어 후 "낙관가 / 균형가 / 회의가 · 협력률 · 읽기 정확도(80% 이상이면 🔮독심술사)"
- 화면 전환 애니메이션, `prefers-reduced-motion` 존중(모든 연출 스킵, 즉시 정산)

## 기술

React + Vite + TypeScript. 외부 UI 라이브러리/이미지 에셋 0 (인라인 SVG + CSS keyframes + Web Audio API). 검증된 게임이론 엔진(전략 8종, 시드 PRNG, payoff, noise) 위에 비주얼 레이어만 얹었다. 모든 시각 상태는 라운드 히스토리에서 파생되는 순수 함수에서 나온다.

## 구조

```
src/core/   순수 도메인 (전략 8종 + 커스텀 전략 컴파일러, payoff, rng, noise)
src/sim/    gameRunner(begin/peek/commit/amend 라운드 진행), match(전략 대 전략),
            evolution(replicator dynamics, 커스텀 종 참가 가능),
            gauntlet(안개의 세계 조우 시뮬 + brute-force 최선 앵커 + 효율 채점)
src/game/   stages(+스테이지 장치), worlds(안개의 세계 4종), progress/worldProgress,
            insight(통찰 경제), twists(텔 파생), mood, trust, combo, outcome,
            world, welfareMessage, analysis, storage
src/ui/     게임 화면 (GamePlay + useRoundPhases 4단계 상태기계, PredictBar, RevealStage,
            ActionChips, TellBadge, BuilderView, EvolutionView/EvolutionChart, Avatar,
            StageMap, StageResult, GardenWorld, WelfareHud, ExchangeStage, RulesModal,
            WorldMap + GauntletView 안개의 세계 ...)
src/audio/  sound (합성 효과음, 콤보 에스컬레이션, 글리치 노이즈버스트)
tests/      115개 (도메인 회귀 + 라운드 분리/재전송 + 통찰/텔 + 커스텀 전략 동치 골든
            + 안개의 세계 결정성/채점 앵커/읽기 분류/메모이즈/베팅 플로우)
```

## 실행

```bash
npm install
npm run dev        # http://localhost:5173/game-of-trust/
npm run test:run   # 회귀 + 게임 로직 테스트
npm run build      # dist/ 정적 빌드
```

## 배경

게임이론 결과를 플레이 경험으로 옮긴 교육용 게임이다. Axelrod(1984) tit-for-tat의 성공 4속성(nice, retaliatory, forgiving, clear), Nowak & Sigmund(1992)의 generous tit-for-tat이 noisy 환경에서 우월하다는 결과, 그리고 협력이 사회후생(2R > T+S, 즉 6 > 5)을 극대화한다는 점이 설계의 뼈대다. 핵심 명제(일방배신은 수확 손실이라는 단조성, AllD가 AllC를 잠식하지만 TFT는 살아남는다는 진화 결과)는 `tests/`가 코드로 잠근다.
